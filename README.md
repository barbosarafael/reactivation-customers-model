# Reactivation Next Purchase

Projeto end-to-end em Databricks para prever se um cliente inativo voltará a comprar nos próximos 30 dias.

## Objetivo

Treinar um fluxo completo de ciência de dados, incluindo ingestão, transformação, feature engineering, modelagem, batch scoring, automação via GitHub e monitoramento.

## Problema

Dado um cliente inativo em uma data de referência, prever se ele realizará uma nova compra nos próximos 30 dias.

## Stack

- Databricks Free Edition
- GitHub
- Python
- PySpark
- Delta Lake
- MLflow
- GitHub Actions

## Dataset

Online Retail II - UCI Machine Learning Repository
